Who Am I?

I am a Full-Stack Software Engineer with a special interest in User Interface Design and Database Internals.

What am I currently obsessed with?

  • Astral — They've got the Midas touch right now, their entire suite of Rust-based tools is pure gold. If you haven't switched to uv yet, you're sleeping. Let uv manage dependencies & virtual environments, ruff for linting, and ty handle your type checking at absurd speeds. Genuinely can't remember what life was like before this, productivity and code quality have increased tenfold.

  • Apache Arrow's PyArrow— I'm usually a heavy Pandas user but PyArrow has kind of stolen the show. The type system is way more rigorous, the in-memory columnar storage is incredibly efficient. It seems Pandas is leaning towards fully adopting PyArrow very soon, but either way it's been a game changer for all tabular work.

  • Apache DataFusion, a query engine written in Rust that I'm definitely obsessed with. When you dig into how it works, you start to appreciate its technical excellence. The gap between "here's what you asked for" and "here's the most efficient way to do it" is where a huge amount of engineering lives — predicate pushdown, projection pruning, join reordering — and DataFusion does all this with a clarity and performance profile that feels like a glimpse at where the next generation of data systems is heading.

  • SQLModel— This one's become my go-to for anything database related. It sits on top of SQLAlchemy and Pydantic which means you get proper data validation basically for free, and writing async APIs that handle database transactions safely feels almost too easy. Less boilerplate, fewer bugs, happier me.

Just a glimpse of some of my favorite tools right now.
